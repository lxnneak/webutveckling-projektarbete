Teknik

Java

Java är ett plattformsoberoende programmeringsspråk som används för allt från webb- och mobilappar till företagslösningar och inbyggda system. Java är objektorienterat och körs på olika plattformar via Java Virtual Machine (JVM).

SQL

Structured Query Language (SQL) används för att hantera och manipulera databaser. Det används för att skapa, läsa, uppdatera och ta bort data i relationsdatabaser genom frågor och kommandon.

HTML

HyperText Markup Language (HTML) är grunden för webbplatser och används för att strukturera innehåll som text, bilder och länkar på en webbsida. Det är byggstenen för webbsidans layout.

CSS

Cascading Style Sheets (CSS) används för att styla HTML-element, som att ändra färger, typsnitt och layouter, för att göra webbsidor visuellt tilltalande och användarvänliga.

Javascript

JavaScript är ett programmeringsspråk som gör webbsidor interaktiva. Det används för att hantera dynamiskt innehåll, som att skapa animationer, hantera användarinmatning och kommunicera med servrar i realtid.
